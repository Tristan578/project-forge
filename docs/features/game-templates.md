# Game Templates

A gallery of fully playable starter projects — select one when creating a new game to skip the blank-canvas setup and start customizing immediately.

## Overview

Game Templates are pre-built scenes with entities, physics, scripts, input bindings, and materials already in place. Each template represents a complete, playable game loop that you can explore, modify, and build on. Templates load directly into the editor as a normal scene, so every element is editable just like something you built yourself.

## Using Game Templates in the Editor

1. Click the **New Scene** button in the toolbar (or press **Ctrl+Shift+N**).
2. The **Template Gallery** opens automatically when creating a new project. You can also open it from the **File** menu.
3. Browse the cards. Each card shows a difficulty badge, an entity count, and genre tags.
4. Click any template card to load it, or click **Blank Project** to start from scratch.
5. The template loads as your current scene — explore the hierarchy to see how it is built.

## What Is Included in Each Template

Each template provides:

- A complete scene with entities already positioned and sized
- Physics and collider configuration on relevant objects
- Scripts for player movement, enemy behavior, and game mechanics
- Input bindings preset for the genre (platformer, FPS, top-down, etc.)
- Materials and lighting tuned for the visual style
- A win/lose condition wired up and ready to test

## Tips

- After loading a template, press **Play** immediately to try it out before making any changes — this helps you understand what the working game looks like.
- The Scene Hierarchy is the best place to explore how a template is structured. Each entity is named descriptively so you can find the Player, enemies, and platforms quickly.
- Beginner templates use simpler scripts and fewer moving parts — start with one of those if you are new to the editor.
